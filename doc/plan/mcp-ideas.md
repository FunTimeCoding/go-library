# MCP ideas

Future MCP servers to build.

## Candidates

PRD — Product Requirements Documents
- Define what to build (problem/requirements)
- https://en.wikipedia.org/wiki/Product_requirements_document

ADR — Architecture Decision Records
- Document how a technical decision was made (context, tradeoffs, consequences)
- https://adr.github.io

SLO — Service Level Objectives
- Define target reliability for a service (e.g. 99.9% uptime)
- https://landing.google.com/sre/sre-book/chapters/service-level-objectives
